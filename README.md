 Hand Gesture Recognition – Finger Counter

A real-time Hand Gesture Recognition project built using **Python, OpenCV and MediaPipe** that detects hands through webcam and counts the number of fingers shown.  
The system can recognize **both hands and count up to 10 fingers in real-time.**

---

 Features
- Real-time webcam detection 
- Detects both hands simultaneously
- Counts 0 – 10 fingers
- Uses Google MediaPipe Hand Tracking
- Beginner friendly Computer Vision project
- Works on Windows with Python 3.11

---

 Technologies Used
- Python
- OpenCV
- MediaPipe
- Computer Vision
